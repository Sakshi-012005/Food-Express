# Food Express 🍔🍕
Food Express is a full-featured online food ordering system that allows users to browse restaurants, place orders, and track deliveries. It includes an admin panel for restaurant management.

## Features
- User registration and login
- Browse restaurants and menus
- Add items to cart and place orders
- Track order status

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
